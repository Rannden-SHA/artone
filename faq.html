General

¿Qué es Art One?
Art One es una plataforma avanzada de Command & Control diseñada específicamente para profesionales de ciberseguridad y pentesting. Proporciona herramientas integradas para pruebas de penetración, análisis de vulnerabilidades y gestión de operaciones de seguridad.
¿Quién puede utilizar Art One?
Art One está diseñado para profesionales de ciberseguridad, pentesters, auditores de seguridad y organizaciones que realizan evaluaciones de seguridad autorizadas. Se requiere verificación profesional para acceder a ciertas funcionalidades.
¿Qué sistemas operativos son compatibles?
Art One es compatible con la mayoría de sistemas operativos modernos, incluyendo Linux, Windows y macOS. La plataforma web funciona en cualquier navegador actualizado.

Licencias y Precios

¿Qué tipos de licencias están disponibles?
Ofrecemos tres tipos de licencias: Community Edition (gratuita), Professional (uso individual) y Enterprise (uso organizacional). Cada una incluye diferentes niveles de funcionalidad y soporte.
¿Cómo funciona la renovación de licencias?
Las licencias Professional y Enterprise se renuevan anualmente. Recibirás notificaciones 30 días antes del vencimiento y puedes configurar la renovación automática para evitar interrupciones.
¿Ofrecen descuentos para educación?
Sí, ofrecemos descuentos especiales para instituciones educativas y estudiantes verificados. Contacta con nuestro equipo de ventas para más información.

Técnico

¿Cómo se instalan los agentes?
Los agentes se pueden instalar mediante nuestro instalador automatizado o manualmente siguiendo la documentación. Soportamos despliegue mediante gestores de configuración como Ansible.
¿Qué protocolos de comunicación se utilizan?
Art One utiliza protocolos seguros y cifrados para todas las comunicaciones. Soportamos HTTP/S, DNS, y protocolos personalizados con capacidades de evasión.
¿Existe una API disponible?
Sí, proporcionamos una API REST completa para automatización e integración. La documentación detallada está disponible en nuestra sección de API.

Seguridad

¿Cómo se protegen los datos?
Todos los datos se cifran en tránsito y en reposo utilizando algoritmos estándar de la industria. Implementamos múltiples capas de seguridad y realizamos auditorías regulares.
¿Qué medidas de seguridad implementan?
Implementamos autenticación multifactor, control de acceso basado en roles, registro de auditoría y monitorización continua de seguridad.
¿Cómo reporto una vulnerabilidad?
Tenemos un programa de divulgación responsable de vulnerabilidades. Puedes reportar problemas de seguridad a través de nuestro proceso documentado en la sección de divulgación responsable.
